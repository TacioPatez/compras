# Carrinho de Compras

Gerenciador de Tarefas

Desenvolvido em React Native

## Objetivo

- O usuário pode criar e editar tarefas;
- Cada tarefa contem um botão para marcar sua “conclusão”;

## Instalação

```
npx react-native init compras
```
